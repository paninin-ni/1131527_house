# 視窗（二）房貸試算器

### 專案簡介:
> 提供直觀且專業的介面供使用者進行房貸還款試算。系統支援「房屋總價」、「自備款比例/金額切換」、「貸款利率」及「貸款年限」等核心參數輸入，並包含選填的「寬限期」設定。具備完善的輸入驗證機制，能攔截非數字、負數或不合理的貸款邏輯（如自備款超過總價）並彈出提示，計算結果皆精確至小數點後兩位並提供千分位格式化顯示。

### 執行說明:
> 1. 輸入數據：在「房屋總價」、「貸款利率」與「貸款年限」欄位輸入數值，並可選填「寬限期」
> 2. 切換機制：透過下拉選單選擇以「百分比 (%)」或「固定金額 (元)」輸入自備款
> 3. 點擊試算：按下「試算」按鈕，程式將依據本息平均攤還公式自動計算
> 4. 查看結果：
> * 貸款總金額：顯示扣除自備款後的實際貸款額
> * 每月應繳金額：呈現寬限期後之每月本息合計負擔
> * 首期資訊：清楚標示第一個月應繳之利息與本金
> * 總支出判讀：顯示還款年限內之「總利息支出」與包含本金在內的「總還款金額」
> 5. 快速重置：按下「清空」按鈕可將所有欄位恢復至預設值，以便進行下一次試算
